opbot
=====

IRC channel operator script for Eggdrop

Requirements
------------

  - Eggdrop 1.6.16+
  - TCL 8.5+

Setup
-----

  1. Install [Eggdrop] 1.6.16+, [TCL] 8.5+.
  2. Copy **ops.tcl** and **util.tcl** to **eggdrop/scripts/**
  3. Copy **ops.sql** to **eggdrop/**
  4. Add `source scripts/ops.tcl` to eggdrop.conf
